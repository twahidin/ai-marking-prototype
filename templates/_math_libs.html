{# KaTeX math-rendering libraries + MathJax-API shim. Opt-in include — only pages that actually render LaTeX/math content pull these in, via `{% block math_libs %}{% include "_math_libs.html" %}{% endblock %}`. Pages without math (hub, dashboard, settings, setup wizard, etc.) skip these 330+ KB of downloads entirely. The shim defines `window.MathJax.typesetPromise` so existing call sites of the form `if (window.MathJax && MathJax.typesetPromise) { MathJax.typesetPromise(...); }` keep working unchanged. On pages without this include, `window.MathJax` is undefined and the guarded calls safely skip. #}
